## Demo GIF Guidelines

When creating your demo GIF:

- **Show the full workflow:** Record → Edit → Share

- **Add captions:** Use a teleprompter script to explain each step
- **Highlight speed:** Emphasize the 60-second pitch creation
- **Show the pitch page:** End with the generated shareable page

- **Reechy itself** (meta!)

- Kap (macOS)
- Loom (export as GIF)
