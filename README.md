## ğŸ¤” The Pitch

We love Loom, but we wanted something open-source, wildly faster, and built specifically for **fast pitch videos**. We didn't want to record a video, download it, open CapCut to trim the bad takes, and then upload it somewhere else just to send it to a client.

Reechy solves this. It's a blazing-fast, open-source video infrastructure that runs entirely in your browser. Record, trim, and generate a beautiful pitch page in seconds.

## ğŸš€ Quick Start

Get your own instance of Reechy running locally in less than 60 seconds:

```bash
# Clone the repository
git clone https://github.com/yerdaulet/fast-pitch.git
cd fast-pitch

# Install dependencies (we recommend pnpm)
npm install
# or yarn install / pnpm install

# Start the development server
npm run dev
```

Visit [`http://localhost:3000`](http://localhost:3000) to start recording immediately.
