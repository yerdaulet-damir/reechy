# Open Graph Images for Reechy

This directory contains social sharing images (Open Graph images) for reechy.cam.

## Required Images

### 1. og-image-default.png (1200x630)
- **Size:** 1200x630 pixels (Facebook/LinkedIn standard)
- **Format:** PNG
- **Purpose:** Facebook, LinkedIn, and other social platforms

### 2. twitter-image.png (1200x630)
- **Size:** 1200x630 pixels (Twitter large card)
- **Format:** PNG
- **Purpose:** Twitter/X summary_large_image cards

## Design Specifications

**Brand Colors:**
- Primary: `#0066FF` (Reechy Blue)
- Background: White to light gray gradient (`#FAFAFA` to `#FFFFFF`)

## AI Generation Prompts

**For Canva/Figma:**
- Use the exact dimensions: 1200x630
- Brand color: #0066FF
- Include product mockup on left side
- Bold headline on right side
- URL at bottom

**For AI Image Generators:**
```
"Open Graph image 1200x630px, modern SaaS product landing page, left side shows clean video recording interface mockup with camera UI, right side has bold headline 'Record a Pitch Video in 60 Seconds', subheadline 'The fastest way to record, edit, and share', small URL 'reechy.cam' at bottom, blue accent color #0066FF, white background, professional clean design"
```

## Tools to Generate

1. **Canva:** https://www.canva.com/ (search "Open Graph" templates)
2. **Figma:** Use 1200x630 artboard
3. **AI Generators:** Midjourney, DALL-E, Stable Diffusion
4. **Online Tools:** https://www.opengraph.xyz/
